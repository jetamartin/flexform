Flex Form Base Styles

This "Base Form Styles Template" primarily illustrates the default styling for HTML Form Elements that is available "out of the box" with Flex Framework (i.e., without the use of special Flex classes). Additionally a few "value add" Flex form features that do require special classes have been applied in this template to improve the overall visual display of the form elements.

Flex form classes applied:

  • flexform-frame - provides a bordered "frame" for the form elements. The "frame" containing the form elements is centered on the page and appropriate paddings and margins are applied to make the frame elements more presentable. This class is applied to a div that wraps the form. The width of the frame can be controlled by applying the approriate Flex grid column sizing class to the "wrapping" div.
  • flexform-title - creates a title for the form and adds a separator below the title. To create the title you must add the title within a header(i.e., h1 - h6) tag and wrap that in a div. The flexForm-title class is applied to the wrapping div. This wrapping div is inserted inside the form.
  • flexform-group - provides vertical spacing (i.e. margin-top/bottom) between form element groups (i.e., label/input element). The form element group must be wrapped in a div. This class is applied to the wrapping div.

Other Flex Classes Used:

  • Flex Grid Classes - the overall alignment and form width is controlled by applying the appropriate Flex Grid classes (e.g., columns and offset). See template below.

Special Notes/Considerations:

  • Input elements (e.g., input[type="text"], textArea, select, etc.) are assigned a default width of 15em's. The textArea is assigned a default height of 5ems.

Flex Vertical Form Template:

This form is an example of a "Vertical Form" format. "Vertical" meaning that the labels for form input elements are "stacked" (vertically) directly above it's corresponding input element (excluding radio buttons and check boxes) rather than being placed adjacent to it on the same line.

Flex form classes applied to achieve this template:

  • flexform-frame - creates a bordered "frame" that surrounds the form elements.
  • flexform-title - creates a title for the form and adds a separator below the title. To create the title you must add the title within a header(i.e., h1 - h6) tag and wrap that in a div. The flexForm-title class is applied to the wrapping div. This wrapping div is inserted inside the form.
  • flexform-group - provides margin separation between form element "groups" (i.e., label/input group). This class is added to the div that wraps each grouping of form elements.
  • flexform-stack - forces the form input element (e.g., button, textarea, text input, etc.) to appear below it's corresponding label (if present). This is class is only applied directly to the form element (not the label).
  • flexform-stack-fullwidth - this class performs the same function as "flexform-stack" class but additionally it forces the input element to fill 100% of the form width (i.e., width=100%;) less any margins and paddings applied to the form. This class should only be to form input element (not on its label).
  • flexform-radio - TBD

Other Flex Classes Used:

  • Flex Grid Classes - the overall alignment and width of the form is controlled by applying the appropriate Flex Grid classes (e.g., columns and offset). See template below.

Special Notes/Considerations:

  • None

Horizontal Form Example:

This is an example of a "Horizontal" form. "Horizontal" means that the form labels are aligned adjacently on the same line as their correspoding form input element (vs being "stacked" on top of each other).
This "Horizontal" template is achieved by using a combination of special Flex Grid System classes, leveraging the "base" default styling inherent in the Flex framework and applying some of the flexform classes:

    The special Flex classes used to achieve this format are as follows:

  • flexform-frame - creates a "frame" for the form and centers the form elements in that "frame".
  • flexform-title - creates a title and separator for between the title and the form contents.
  • flexform-group - provides margin separation between form element groups (i.e., label/input group).
  • flexform-labelAdjacent-right - causes the label to align adjacent to it's corresponding input element and aligned on the right side of the label "column" (i.e., "text-align: right")
  • flexform-labelAdjacent-left - causes the label to align adjacent to it's corresponding input element and aligned on the left side of the label "column" (i.e., "text-align: right")

Other Flex Classes Used:

  • Flex Grid Classes - Besides being used to control the overall width and alignment of the form, the Flex Grid classes are used extensively within the form tag to establish the layout of the "Horizontal" frame. Specifically you will need to create a column for the label column (by applying a column width to each label) and a column for the input form elements (by applying a column width to each input element). You may also need to use column offsets for certain input elements like checkboxes and radio buttons given that the label typically is on the right side of the input element and you would typically want both it's label and it's input element (i.e., checkbox/radio button) to both appear entirely in the input form element column. (See example in template).
